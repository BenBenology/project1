# Skills 说明

这个文件描述当前项目实现所需的关键能力，以及每项能力现在处于什么阶段。

## 当前已具备的能力

### 1. 查询任务编排

- 能接收搜索词和查询类型
- 能创建任务并返回任务 ID
- 能查询任务状态和进度

### 2. mock 文档生成

- 能生成财报、公告、新闻、深度文章四类数据
- 能输出统一结构的标题、来源、摘要、标签和附件
- 能根据查询词生成稳定的 mock 结果

### 3. 前端快速验证

- 能用 Streamlit 快速搭建交互页面
- 能发起任务、轮询状态、展示结果
- 能在不引入正式前端工程的情况下验证产品流程

## 即将补齐的能力

### 1. 真实数据抓取

- 接入官方 IR 页面
- 接入公告站点
- 接入新闻或 RSS 来源
- 做限频、重试和失败隔离

### 2. 数据解析与清洗

- 抽取发布时间、标题、正文、附件链接
- 做去重和标准化分类
- 给财报和新闻统一标签

### 3. 数据持久化

- 用数据库保存任务和文档
- 用对象存储保存 PDF 或缓存下载文件
- 用缓存提高重复查询效率

### 4. 智能摘要

- 接入大模型做摘要和重点提炼
- 对长文和财报做结构化摘要
- 控制摘要成本和错误率

## 当前能力边界

这个项目现在还不具备以下能力：

- 真实在线爬虫
- 多用户权限系统
- 交易建议或自动下单
- 高可用生产部署
- 全量内容版权合规策略

## 能力升级路线

1. mock 数据
2. 半真实数据源
3. 真实抓取 + 落库
4. 摘要和标签自动化
5. 订阅、推送和个性化
